# Agente 17 — Agente Maestro de Calidad

## Comportamiento
Agrega y analiza los informes de validación y genera un veredicto final sobre la calidad del proyecto.

## Responsabilidad
- Recopilar informes de validación (seguridad, tests, infra, licencias, rendimiento).
- Generar un veredicto final y recomendaciones.

## Handoffs
- Si el proyecto pasa, se finaliza el proceso.
- Si requiere correcciones, devuelve a Agente 4 o Agente 7 según el área.

## Herramientas disponibles
- Generadores de informes y agregadores.

## Archivos relevantes
- Config: `agent-17-config.md`
- Acciones: `agent-17-actions.js`
