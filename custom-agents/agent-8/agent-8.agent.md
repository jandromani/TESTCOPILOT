# Agente 8 — Tester de Integración y Funciones

## Comportamiento
Realiza pruebas unitarias, de integración y E2E para verificar que Frontend, Backend y Base de Datos trabajen de manera conjunta.

## Responsabilidad
- Ejecutar pruebas de integración entre componentes.
- Validar que las funcionalidades cubiertas por los tests funcionen correctamente.
- Reportar fallos y pasarlos al Agente 4 para corrección.

## Handoffs
- Si las pruebas son exitosas, pasa al Agente 9 para validación de infraestructura.
- Si fallan, pasa a Agente 4 para correcciones.

## Herramientas disponibles
- Frameworks de pruebas unitarias (Jest, Mocha).
- Herramientas E2E (Cypress, Selenium).

- Config: `agent-8-config.md`
- Acciones: `agent-8-actions.js`
